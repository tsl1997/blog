jQuery基础（完结篇）

发表于 2022-04-27 更新于 2023-02-01 分类于 JS
前端学习笔记

jQuery内容：

CSS样式

1：链式写法

对象.css(“属性”,”值”).css(“属性”,”值”).css(“属性”,”值”)

2：大括号写法

对象.css({“属性”:”值”,”属性”:”值”,”属性”:”值”})

3：Class写法

  • 先定义类名CSS样式（建议在css文件里面写）
  • 再用 对象.addClass(“class”);添加样式
  1. addClass() 添加样式
  2. removeClass() 删除样式
  3. toggleClass() 交替样式
  4. hasClass() 判断样式

事件

页面加载事件

$(function($) { }); //网页结构加载完成开始执行

键盘事件

keyup() 键盘按钮弹起触发事件
keydown() 键盘按钮按下触发事件
keypress() 键盘按下或弹起触发事件

焦点

focus() 聚焦触发事件
blur() 焦点离开触发事件

鼠标

click() 单击事件
dblclick() 双击事件
mouseover() 鼠标停留
mouseleave() 鼠标离开
hover(停留函数,离开函数) 一个方法搞定鼠标的停留和离开
toggle(函数1,函数n) 点击交替调用函数

事件处理：

常规处理：

bind(“事件”,函数) 绑定事件 该方法在3.0后被弃用，请用on()方法代替
unbind(“事件”) 取消绑定 该方法在3.0后被弃用，请用off()方法代替
one(“事件”,函数) 一次性

委托事件：

delegate(“选择器”,”事件”,函数) 可以动态的给未来产生的选择器(标签)委托任务
该方法在3.0后被弃用，请用on()方法代替
undelegate(“事件”) 可以把delegate委托的事件给删除掉
该方法在3.0后被弃用，请用off()方法代替

更多内容请查看帮助手册或者在线文档：https://jquery.cuishifeng.cn/
