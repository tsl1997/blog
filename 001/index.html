JS预习快速总结

发表于 2022-04-19 更新于 2023-02-01 分类于 JS
前端学习笔记

运行环境：任何浏览器
导入方式：

  1. 内部
  2. 外部

F12的基本使用：
元素：查看html和css样式
控制台：调试JS
源代码：可以打断点debug
网络：获取网络提交的数据
应用程序：保存了一些缓存和Cookie

基本数据类：
var 变量名=值；
可以理解为Object，而且他会自动类型转换。

  • 不区分浮点数和整数
  • 只有字符串，没有char类型
  • 除了null外，还有 undefined(未定义) 和NaN（不存在数值）

运算符：
关系运算符在比较等于===绝对等于，因为等于==会自动转换，造成数值1==”1”的结果是true

流程控制：
基本一样

数组：
因为类型为object，所以可以放入任何类型。
类型一般使用()或者[]，一般建议使用后者。

对象：
可以直接创建对象，并在大括号后写入属性，还可以动态的修改属性的值和属性本身。

继承：
默认的JS代码会继承一些已有的方法。如数组会继承一些数组类的基本操作方法，诸如排序等。
