nginx安装

发表于 2023-01-16 更新于 2023-01-17 分类于 Linux
腾讯云服务器测试安装nginx

nginx是一个优秀的静态资源服务器，也可以用作于反向代理、负载均衡。

1：安装nginx

1.1：安装Windows版本

为什么不去Linux上，因为不想开虚拟机或服务器，懒或穷。

官网下载地址：http://nginx.org/en/download.html

选择对应的系统版本下载即可。一般用稳定版本较好。

MainLine version 主线版本

Stable version 稳定版本

Legacy versions 历史版本

1.2：目录简介

下载好之后解压打开，有以下目录：

  • conf 配置目录 重要配置文件nginx.conf
  • html 文件目录 存放网页资源 默认index.html 、50x.html
  • logs 日志目录 程序运行后有error.log、access.log
  • temp 临时目录 存放临时文件

1.3：启动命令

双击nginx.exe程序。

2：Linux安装

1：安装插件

gcc环境，编译需要：
yum -y install gcc
安装PCRE库，nginx的HTTP模块依靠他：
yum install -y pcre pcre-devel
安装zlib:
yum install -y zlib zlib-devel

2：下载nginx安装包

因为nginx非常小2M左右，所以无需提前下载到本地。
下面指令可以自行修改官网对应的版本：

wget http://nginx.org/download/nginx-1.22.1.tar.gz

3：解压

tar -zxvf nginx-1.22.1.tar.gz

4：编译安装

cd nginx-1.22.1
./configure
make && make install

5：启动nginx

cd /usr/local/nginx/
sbin/nginx
