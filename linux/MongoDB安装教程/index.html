MongoDB安装教程

发表于 2022-12-17 更新于 2022-12-30 分类于 DB
作者@XS——mongo环境安装

一个分布式文件存储的数据库，属于非关系型的数据库。
mongo和mysql的对比图：

Linux环境安装MongoDB：

1：先装Docker

yum install docker
docker -v

2：基本命令：

启动：systemctl start docker
停止：systemctl stop docker
查看：systemctl status docker

3：下载镜像

docker search 镜像名
docker pull 镜像名:版本号

docker pull mongo:4.4

4：启动容器

docker run -di –name=mymongo -p 27017:27017 mongo:4.4

5：进入容器

docker exec -it mymongo bash

6：停止和删除容器

查看运行容器：docker ps
停止运行容器：docker stop 容器id
删除容器：docker rm 容器id

7：带验证的运行mongo容器

docker run -di –name=mymongo -p 27017:27017 mongo:4.4 –auth

Windows安装MongoDB:

1：下载

下载地址：
https://www.mongodb.com/try/download/community

建议版本也选择和本教程对应的。

2：安装

建议安装到纯英文目录，建议自定义。
本版本已经帮你创建了log和data目录。

去掉默认选中：

3：启动

在bin目录下启动下面两条命令（请按顺序分别开两个窗口）：

mongod –dbpath C:\JavaCode\Tools\WebHTTP\mongoDB\mongo\data\db
mongo
