最简单的翻墙姿势：利用Github获取大量免费节点。

发表于 2021-08-25 更新于 2021-12-06 分类于 教程
翻墙姿势多，GitHub人才济济

之前写过一篇，结果挂了，翻车是很正常的，毕竟这种环境下，科学上网都是活不久的。

现在是某个UP主给我的启发。

就全程用国内的IP来演示，如何寻找，如何获取，如何翻墙？

1：访问Github

https://github.com/

我们就不去介绍这个同性交友网站了，来看看该怎么样操作把？

我们选择第二个free(免费） 的仓库看看。

可怜我那为数不多的几个英语单词，终于是用上了。

2：查看他人的仓库和说明

https://github.com/freefq/free

一个要看人气，另外一个要看更新的频率。

这个仓库真的不错，每日更新，而且节点大多数都是可用的。

我们可以看到，在Android,ios,windows都可以使用，我们暂时先来用win的。

如何不会也可以去查看他官方的文档，在上图已经标出了。

3：下载软件和订阅方法下载软件：

Windows的下载链接：https://github.com/2dust/v2rayN/releases/download/3.27/v2rayN-Core.zip

下载好，解压后，得到这样的一个文件夹，双击v2rayN.exe运行。

进入软件后，我们是没有任何节点的。 设置订阅白嫖时间到了，我推荐那么使用订阅的方式，因为当仓库主再次更新了之后，你只需要更新订阅即可获取最新的节点。

点击订阅，订阅设置。

我们添加上图这条订阅：

重要的是地址（URL），备注随便写就可以了。

https://raw.fastgit.org/freefq/free/master/v2

更新订阅：

然后重点来了，我们回到主页面，点击订阅，更新订阅:

这样就已经订阅成功了。

测试链接：

我们随便选择一个节点，然后按住Ctrl+A（全选），然后点击鼠标右键，选择*测试真链接延迟。（Ctrl+R）*

一会儿我们就可以看到各个节点的延迟情况，我们选择一个有延迟的即可。

链接节点：

我们随便选择一个节点，然后右键，设置为活动服务器即可，也可直接按下回车键（enter）。

这样就设置成功了。 开启代理： 这个只需要做一次就好。

我简单说一下，全局和PAC的区别。

全局代理所有的流量都会走服务器，在访问一些国内的网站的时候会反应较慢。

PAC则绕过了大陆的网站，在国内就用你自己的IP（速度快），在访问国外网站的时候就走代理。节省流量是一个方面，另外一个在访问国内网站是体验会好一些。但是对于一些不常见的国外网站可能会无法访问，这时候请切换为全局代理即可。 所以，在一般的时候，都是使用PAC代理的。

当然，如果速度不行就换一个节点，只要是可以使用的节点都行。

速度还是不错的，可能是我这边的条件不怎么样，就只能测到这个速度了。

4：安卓的使用

安卓下载链接：https://github.com/2dust/v2rayNG/releases/download/1.4.12/v2rayNG_1.4.12_arm64-v8a.apk

我们将下载好的APK安装到手机/模拟器。

打开后，我们点击左边，然后选择订阅设置。

我们还是一样的导入订阅的链接： https://raw.fastgit.org/freefq/free/master/v2

我们添加一条订阅即可。 当然，你也可以添加多条订阅。

我们选择三个点的图标，选择更新订阅。 我们还是可以选择全部测试配置真链接（看上图）。

这样我们就会获得全部的测试结果，然后选择延迟低的就好了。

当然，我们也可以设置，在里面选择路由设置（预定义规则）绕开大陆地址，效果也是一样的。

最后选择小V链接即可。
