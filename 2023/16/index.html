AAVE测试版GHO教程——Mint无数测试币

发表于 2023-09-21 更新于 2024-01-18 分类于 测试网
新人可以练练手，过过瘾，最起码看着舒服

https://gho.aave.com/

GHO是著名抵押借贷项目AAVE的测试版本，目前提供Goerli和Sepolia两个市场。

有部分人建议去交互，我在交互时发现一个好玩的。

接下来的内容可能需要你自备一点DEFI知识。（基础概念不解释）

教程开始：

1：连接钱包领水

右上角连接钱包，然后选择水龙头Faucet。

我们从四个里面随便选择一个，比如上图我选择的是AAVE测试币，然后确定Faucet AAVE。

授权合约交互图片就省略了，无脑点击即可。

我们成功领取了100个AAVE测试币，但是我发现操作居然是Mint。

如果大家Mint过NFT就应该明白我接下来要干什么。

2：Mint测试代币

https://sepolia.etherscan.io/tx/0x589bb168ade5069096a66f7918a82a43f5fd125f60d49e534e0d8491be3a0728

把这个页面保留，里面还有数据用得到。

打开交易地址，新页面再打开合约地址：

看图操作。

把之前的页面下面的数据，得到下列信息：

Function: mint(address token,address to,uint256 amount)

MethodID: 0xc6c3bbe6
[0]: 000000000000000000000000f9abbc6e64f3cb3cb237ebcab95a252365bbd0d0
[1]: 000000000000000000000000f0438d3836d312641e578189ab1990816cec811d
[2]: 0000000000000000000000000000000000000000000000056bc75e2d63100000

我们点击Decode Input Data转换一下：

# Name Type Data
0 token address 0xf9aBBc6E64f3Cb3cb237EbcAB95A252365BBd0D0
1 to address 0xF0438d3836D312641e578189AB1990816Cec811d
2 amount uint256 100000000000000000000

token就是你要mint的代币合约地址。

to填写你的钱包，amount是数量，一般默认为18位小数。

我们把数据复制到上面的写入合约中，把数量再加几个零。

https://sepolia.etherscan.io/tx/0xb0aec3753f83b339de4892f6c28908d7abc0e24d91310d641cc2164628766263

Mint成功了。

我们可以去GHO愉快玩耍了。
